# Clima Libre 🌤️

Una hermosa app de clima minimalista y privada, sin anuncios ni trackers.

**Características:**
- Ilustraciones únicas con una chica representando cada condición climática
- Unidades europeas (Celsius, km/h, hPa, mm)
- Diseño bonito y moderno
- Totalmente offline-capable (PWA)
- Datos de Open-Meteo (100% gratis y privado)

## Despliegue en Vercel
1. Importa este repositorio en Vercel
2. ¡Listo! Se despliega automáticamente.

Hecha con ❤️ usando ilustraciones personalizadas.
